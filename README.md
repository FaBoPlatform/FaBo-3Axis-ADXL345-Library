# FaBo3Axis-ADXL345-Library

This is a library for the FaBo #201 3AXIS I2C Brick.

[#201 3AXIS I2C Brick](http://fabo.io/201.html)

ADXL345 is 3-Axis Digital Accelerometer sensor.
